# Spire Slayer

spireSlayer is an AI designed to beat Slay the Spire 2. Currently, it does absolutely nothing but hopefully that will change. 

You can go [here](notes/plan.md) to view the progress.


## Running The Ai

### Install dependencies:
```
py -m pip install -r requirements.txt
```

### Run the project:
```
py src/main.py
```
